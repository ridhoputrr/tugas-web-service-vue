# Web Service Application with Vue 3

## Overview

This project demonstrates the implementation of REST API integration in a Vue 3 application using Axios for HTTP communication and Vue Router for client-side navigation. The application is designed to retrieve, display, and manage user data through API requests while applying modern frontend development practices.

## Features

* Display a list of users retrieved from an API
* View detailed information for individual users
* Create new users through a form interface
* Implement HTTP GET and POST requests using Axios
* Client-side routing with Vue Router
* Responsive and user-friendly interface

## Technologies Used

* Vue 3
* Vite
* Axios
* Vue Router
* JavaScript (ES6+)
* HTML5
* CSS3
